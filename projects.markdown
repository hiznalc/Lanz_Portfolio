A comprehensive health information system designed to streamline patient data management and improve healthcare delivery for the local community.

### Features

- Patient records management
- Health data tracking
- Report generation
- User-friendly interface for health workers

### Technologies Used

- HTML & CSS
- JavaScript
- PHP
- MySQL Database

### Impact

This system helps RHU Kawit efficiently manage patient information and deliver better healthcare services to the community.
